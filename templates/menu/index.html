Huffman coding provides codes to characters such that the length of
the code depends on the relative frequency or weight of the
corresponding character. Huffman codes are of variable-length,
and without any prefix (that means no code is a prefix of any other).
Any prefix-free binary code can be displayed or visualized as a binary
tree with the encoded characters stored at the leaves.
